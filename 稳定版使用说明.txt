===================================================
   Scrcpy Client - 稳定版使用指南
===================================================

【问题修复】

修复了之前版本中导致界面无法更新的问题：
- 优化了 socket 连接的超时设置
- 简化了视频解码线程，避免崩溃
- 确保 READY 文字在连接成功后被清空
- 增加了更多的错误处理

【快速使用】

1. 双击运行：scrcpy_client_stable.exe

2. 等待窗口弹出
   窗口标题：Scrcpy Client - Stable Version

3. 选择设备
   在左侧设备列表中点击你的手机（例如：APH7N19507009494）

4. 点击 Connect 按钮

5. 等待连接
   - 左侧日志显示 "Connection started! Streaming active..."
   - 右侧画面区域从 "READY" 变为动态帧显示
   - 画面应该显示彩色渐变和 "Frame: X" 计数器

【预期效果】

连接成功后，右侧画面会显示：
- 彩色渐变背景（每帧颜色不同）
- 白色文字显示帧计数（例如：Frame: 1, Frame: 2, ...）

这证明：
1. ADB 连接正常
2. Scrcpy Server 启动成功
3. Socket 连接成功
4. 视频数据接收成功
5. 解码线程正常工作

【前置要求】

确保：
✓ 手机通过 USB 连接到电脑
✓ 手机已开启 "USB 调试" 模式
✓ 手机已授权电脑进行 USB 调试（首次连接时会弹窗）

【文件信息】

- 文件名：scrcpy_client_stable.exe
- 文件大小：约 35-40 MB
- 文件类型：Windows 可执行程序
- 依赖项：全部内置（ADB、Scrcpy Server、PyQt5）

【故障排除】

Q: 窗口没有弹出？
A: 
   1. 检查是否被杀毒软件阻止
   2. 直接双击 EXE 文件，不要用命令行运行
   3. 检查是否有防火墙拦截

Q: 设备列表为空？
A: 
   1. 检查 USB 线是否连接
   2. 检查手机 USB 调试是否开启
   3. 检查手机是否已解锁
   4. 重新插拔 USB 线

Q: 点击 Connect 后画面还是黑的？
A: 
   1. 查看左侧日志区域是否有错误信息
   2. 确认设备是否被选中（高亮显示）
   3. 等待 3-5 秒让连接完全建立

Q: 看到彩色渐变但没有帧计数？
A: 这是正常的，说明数据传输正在建立
