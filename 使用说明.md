# WiFi手机投屏 - 完整使用指南

## 项目文件说明

### 主程序文件
- `wifi_mirroring_final.py` - 最终完善版GUI程序（功能完整）
- `wifi_mirroring_app.py` - 简化版程序

### 打包文件
- `一键打包.bat` - 双击即可自动打包为EXE
- `requirements.txt` - 运行依赖
- `requirements-build.txt` - 打包依赖

## 三种使用方式

### 方式1：直接运行Python（最简单）

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python wifi_mirroring_final.py
```

### 方式2：打包为EXE（推荐分发）

1. 双击运行：`一键打包.bat`
2. 等待3-5分钟
3. 生成的文件：`WiFi_Mirroring.exe`
4. 双击EXE即可运行，无需Python环境

### 方式3：手动打包

```bash
pip install pyinstaller pyqt5 qasync opencv-python numpy mss
python -m PyInstaller --onefile --windowed --name "WiFi_Mirroring" wifi_mirroring_final.py
```

## 使用步骤

### 桌面屏幕投屏
1. 运行程序
2. 点击"启动服务器"
3. 在手机上用VLC打开显示的RTSP地址
4. 开始投屏！

### Android投屏
1. USB连接手机
2. 在手机上启用"开发者选项"和"USB调试"
3. 点击"连接Android设备"
4. 开始投屏！

## 常见问题

### Q: 提示缺少模块
A: 运行 `pip install -r requirements.txt`

### Q: 打包失败
A: 确保安装了pyinstaller: `pip install pyinstaller`

### Q: 运行时闪退
A: 检查是否安装了所有依赖，查看同目录下的log文件

## 注意事项

1. 确保手机和电脑在同一WiFi网络
2. 防火墙可能阻止连接，需要允许程序通过防火墙
3. 建议使用VLC播放器接收投屏

## 技术支持

如遇到问题，请查看日志文件：`wifi_mirroring.log`
